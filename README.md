# Mitusko-Intellij Plugin
***
### Version 1.0 for Mitsuko 1.0
## Features
* Syntax highlighting for functions, event links, items, and pack.msk
* Autocomplete for functions, items, and pack.msk
* Block Folding
* Mild error detection
